# 林听雪

## 基本观察

- 主人公左胸锁骨下方有仇痕，遇相关线索时会发热甚至灼痛 [ch-1]
- 主人公持有断剑，剑刃有师父留下的望气印记，出鞘时泛萤光 [ch-1]
- 主人公来自碑谷，已在碑谷生活十年 [ch-1]
- 主人公曾多次杀死玄剑宗的探子，尸骨埋在断剑碑下 [ch-1]
- 主人公头上戴着一支玉钗，是师父的遗物，钗头刻半开雪莲 [ch-1]
- 主人公的剑法比郑寒锋更快，能用断剑压偏对手剑身 [ch-1]
