# 郑寒锋

## 基本观察

- 郑寒锋是雪夜城守将，左脸有一道剑痕，是冻伤后愈合的疤痕 [ch-1]
- 郑寒锋曾在十年前雪夜巡逻时听见城里有人喊叫，没有回头 [ch-1]
- 郑寒锋知道师父的死亡情况：他看见一个女人躺在雪地，箭穿胸口，手握玉钗 [ch-1]
- 郑寒锋认为师父已死，但玉钗细节暗示尸体可能并非师父本人 [ch-1: inferred]
- 郑寒锋转交沈师傅铸的新剑鞘，并传达'另半部剑谱不在他那里' [ch-1]
