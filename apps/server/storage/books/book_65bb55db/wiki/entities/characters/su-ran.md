# 苏染

## 基本观察

- 苏染声称三年前就进了雪夜城，有熟客，但实际七天前从碑谷方向来，自称药商 [ch-1]
- 苏染曾作为药师被骗送'安神散'到一处宅邸，导致宅主死于忘川散 [ch-1]
- 苏染花了三年找出当年雇她的人，线索指向玄剑宗，并盗出账簿 [ch-1]
- 苏染曾摸进玄剑宗外门，盗出账簿上有'碑谷'被涂抹的记录 [ch-1]
- 苏染在南边七里外的破庙等主人公 [ch-1]
