# 第 1 章：第一章·雪夜抵城

## 一句话概要
主人公持城引抵达雪夜城，与守将郑寒锋对峙后获准入城。郑寒锋透露师父十年前死于玄剑宗之手，但玉钗细节暗示尸体可能并非师父。郑寒锋转交沈师傅铸的新剑鞘，并告知另半部剑谱不在他处。主人公决定去找苏染对质。
